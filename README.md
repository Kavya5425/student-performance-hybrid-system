# Student Performance Prediction (Hybrid System)

## Description
This project uses a hybrid intelligent system combining Fuzzy Logic and Neural Networks (ANFIS) to predict student performance.

## Inputs
- Attendance
- Assignment Marks
- Test Marks

## Output
- Performance Level (Poor, Average, Good)

## Methodology
- Fuzzy logic defines rules
- ANFIS applies neural learning
- Membership functions are optimized

## Tools Used
- MATLAB
- Fuzzy Logic Toolbox
